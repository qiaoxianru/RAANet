# RAANet: Radial Attenuation Attention Network for Image Classification

This repository provides the official PyTorch implementation of the paper:

> **Radial Attenuation Attention for Image Classification**  


The code implements the **RAA module** (Radial Attenuation Attention), which combines **RADConv** (Radial Attenuation Directional Convolution) and **FAM** (Feature Attenuation Modulation) to mimic the center‑peripheral attenuation mechanism of biological vision. The network consistently improves classification accuracy on CIFAR‑10/100, SVHN, Imagenette, Imagewoof, and ImageNet‑1K.
